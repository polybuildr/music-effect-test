Typing test

  • You will be presented with a paragraph and a box below it to type in.
  • You must type the given paragraph correctly as fast as possible in the box below.
  • A timer will appear below the box to show how long you are taking.
  • Once you click the button below (or press enter), the test will load and the timer will have begun. You can start typing immediately.

Example:

{{elapsedSeconds.toFixed(1)}} seconds!
Next test in {{secondsLeftForNextParagraph}} seconds...
